## 1. System Dependencies (系统依赖)

在开始构建之前，请确保您的开发环境（厨房）已安装以下依赖包（食材）：

### Core Modules (核心组件)
*   **牛肉沫**: 1斤 (建议根据 Patch Notes 选择部位)
*   **番茄**: 2-3个 (切成小丁，越小越好)
*   **洋葱**: 1/2个 (切丁)
*   **大蒜**: 6-7颗 (拍碎)

### Runtime Libraries (运行时调料库)
*   **橄榄油**: 喷3下 (低功耗模式，减脂友好)
*   **番茄酱**: 2圈
*   **白糖**: 1勺
*   **高度白酒**: 1瓶盖 (兼容性提示：无白酒可用1勺料酒代替)
*   **生抽**: 1勺
*   **老抽**: 0.5勺
*   **盐**: 1勺 + 适量 (用于后期调试)
*   **黑胡椒粉**: 1勺 + 少许 (用于后期调试)
*   **海盐**: 少许
*   **淀粉**: 1勺
*   **水**: 约半碗 (用于冲洗容器及调整水位)

## 2. Configuration (参数配置)

为了优化构建流程，我们需要预先编译两个配置文件（碗汁）。

*   **Config_Set_A (番茄基底)**:
    *   在一个大碗中放入：切好的**番茄丁** + 挤入 **2圈番茄酱** + **1勺白糖**。
    *   *状态*: 备用。

*   **Config_Set_B (增稠与调味)**:
    *   在另一个碗中放入：**1勺生抽** + **1勺盐** + **1勺黑胡椒粉** + **0.5勺老抽** + **1勺淀粉**。
    *   *Action*: 搅匀备用。

## 3. Build Pipeline (构建过程)

请严格按照以下步骤执行构建，注意关键节点的**加粗**参数。

1.  **Initialize Environment (环境初始化)**:
    *   起锅，锅里**喷3下橄榄油**。
    *   *Note*: 油不用很多，牛肉沫炒制时会析出油脂。

2.  **Load Pre-requisites (加载前置依赖)**:
    *   油热后，下入**洋葱丁**和**蒜末**。
    *   炒香。

3.  **Process Core Data (处理核心数据)**:
    *   倒入**牛肉沫**，**大火**炒散、炒熟、炒透。
    *   中间烹入 **1瓶盖高度白酒** 去腥。
    *   *Checkpoint*: 直到牛肉沫油脂析出，呈现**油亮油亮**的状态。

4.  **Merge Config_A (合并番茄配置)**:
    *   将 **Config_Set_A (番茄碗汁)** 全部倒入锅中。
    *   加半碗水涮一下碗里的残留番茄酱，倒入锅中。
    *   *Requirement*: 锅里的水差不多刚好能**稍没过牛肉沫**。

5.  **Compilation (编译/焖煮)**:
    *   先**中火**把番茄丁炒一炒，加少许**黑胡椒海盐**。
    *   盖上盖子，转**中小火**焖煮。
    *   **Time**: **5-10分钟**。
    *   *Process Goal*: 把番茄完全炖软烂，变成番茄汤汁。
    *   *Warning*: 全程**盯着锅里的水量**，防止Stack Overflow (烧干)，也不要太多水导致风味稀释。

6.  **Verify & Boil (验证与沸腾)**:
    *   开盖，确认番茄丁已完全炖化成汤汁并融在牛肉沫里。
    *   转**大火**，将汤汁煮至**大沸**状态。

7.  **Inject Config_B & Finalize (注入增稠配置与终局)**:
    *   在沸腾状态下倒入 **Config_Set_B (淀粉料汁)**。
    *   保持**大火**，翻拌均匀。
    *   *Debugging*: 此时尝一下咸淡，根据口味补点**盐** or **黑胡椒**。
    *   观察酱汁收浓，直到牛肉沫**红亮红亮**，无多余油水，呈现**黏糊红亮**的酱汁状态。

8.  **Release (发布)**:
    *   关火，出锅。

## 4. Deployment (部署方案)

本产品支持热部署，无需冷却，直接上线。

*   **Scenario A: 标准米饭环境 (Recommended)**
    *   直接挖几勺酱与米饭拌匀。
    *   *Optional Add-ons*: 煎一个半熟蛋 + 烫青菜。
    *   *User Experience*: 酸甜番茄味与浓郁牛油香气包裹每一粒米饭。

*   **Scenario B: 面条环境**
    *   选细面条，煮好后过凉水。
    *   擦点黄瓜丝，拌入多多的酱汁。
    *   *User Experience*: 黏黏糊糊，清爽与浓香并存。

*   **Scenario C: 低碳/减脂环境**
    *   洗几片生菜叶。
    *   包着牛肉酱吃，可加点小葱。

## 5. Patch Notes / Optimization (版本优化与日志)

整合自社区评论区（Code Review），可根据实际需求Cherry-pick以下特性：

*   **Feature Request: 批量处理**
    *   可以一次做一大锅，使用冰格分装冷冻。煮意面时直接化开浇在面上（类似 Bolognese）。
*   **Performance Boost: 原料升级**
    *   推荐使用**新疆产番茄罐头**（带皮番茄丁或整只去皮）代替新鲜番茄，味道更鲜美。
    *   番茄酱建议选用配料表只有番茄的新疆番茄酱。
*   **Refactoring: 肉质选择**
    *   **牛腩**: 脂肪多，更香（User: `知乎用户jyJGBk`）。
    *   **牛里脊**: 味道好但颜色可能发暗，建议加点黄油改善色泽（User: `一只猫的救赎`）。
*   **Bug Report: 色泽问题**
    *   如果成品颜色发暗不红亮，可能是老抽过多或牛肉部位偏瘦，可尝试调整老抽用量或增加油脂（黄油）。
